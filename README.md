# aws-static-website-hosting
This project demonstrates how to host a static website using Amazon S3.

## Project Overview

This project demonstrates how to host a static website using Amazon S3. The website is publicly accessible through an S3 Static Website Hosting endpoint and serves HTML and CSS files without requiring a web server.

## Features

- Static website hosted on Amazon S3
- Public access configured using Bucket Policies
- Static Website Hosting enabled
- Custom file
- Accessible through a public S3 website endpoint

## Technologies Used

- Amazon S3
- AWS IAM
- HTML
- Git
- GitHub

## Implementation Steps

1. Created an Amazon S3 bucket.
2. Disabled Block Public Access settings.
3. Uploaded website files.
4. Enabled Static Website Hosting.
5. Configured Bucket Policy for public read access.
6. Tested website accessibility through the S3 website endpoint.
7. Managed project files using Git and GitHub.
